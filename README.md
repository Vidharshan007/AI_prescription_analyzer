# AI Prescription Analyzer 🏥

An AI-based web application that extracts text from doctor prescriptions using OCR (Tesseract) and displays the results through a Flask web interface.

## 🚀 Features
- Upload prescription images
- Extract text using Tesseract OCR
- Simple and clean UI using Flask
- Handles image processing using OpenCV

## 🛠️ Tech Stack
- Python
- Flask
- OpenCV
- Tesseract OCR
- HTML

## 📂 Project Structure
- app.py → Backend logic
- templates/ → HTML files
- static/ → CSS/JS
- uploads/ → Stores uploaded files

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
